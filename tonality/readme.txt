В качестве решения прилагается архив “Lebedev_tonality.zip”, содержащий файл с программой prog.py, файлы с текстами для тестирования в папке examples, отчёт, csv-таблицу со словарем тональности и отзывы в папке reviews.
Для работы программы необходимо:
1. Наличие установленных модулей nltk, pymorphy3 и pandas;
2. В дной папке с программой должны находиться папки examples и reviews;
3. Необходима версия Python от 3.10.
